## Windows

### Dành cho trình đọc màn hình NVDA

Để sử dụng RHVoice với NVDA, trước tiên bạn hãy cài đặt add-on RHVoice chính.

{{%nvdaAddonLink "Tải xuống add-on"%}}.

Phiên bản này hỗ trợ NVDA từ 2021.1 đến 2026.1.

Sau đó, truy cập các liên kết ngôn ngữ trên [trang này]({{<relref "languages">}}) để tải xuống và cài đặt add-on giọng nói tương ứng.

### Dành cho các chương trình khác trên Windows

Bạn cũng có thể sử dụng RHVoice với các ứng dụng Windows khác, chẳng hạn như trình đọc màn hình JAWS, nếu chúng tương thích với Microsoft Speech API phiên bản 5 (SAPI 5).

Truy cập các liên kết ngôn ngữ trên [trang này]({{<relref "languages">}}) để tải xuống và cài đặt giọng nói. Không cần cài đặt thêm phần mềm nào khác vì trình cài đặt giọng nói đã bao gồm mọi thứ cần thiết.

## Android

Cài đặt ứng dụng RHVoice từ {{%googlePlayLink "Google Play"%}}.

Sau đó, mở ứng dụng và tải xuống các giọng nói bạn muốn sử dụng.
